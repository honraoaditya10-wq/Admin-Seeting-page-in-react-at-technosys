# Admin Setting Page (React)

This project is an **Admin Settings Page** built using **React**, **TypeScript**, and **Tailwind CSS**.  
It was developed as part of the **JobConnect web app project at Technosys**.

## Features
- Admin settings dashboard
- Role & permission management
- Security settings
- Profile editing
- Policy and terms management
- Activity logs

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Create React App
